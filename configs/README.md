# 配置文件(Configs)

#### 文件格式有

```
1、.ini
2、.json
3、.xml
4、.yaml
5、.toml
6、.properties
7、.HOCON
8、.plist
9、.cfg
10、.xsl
11、.config
12、.conf
13、.property
14、.txt
15、.sys
16、.inf

一般存储二进制格式数据：
17、.data
18、.data

与数据库有关的：
19、.database
20、.db

....

可以存储的数据复杂度：xml>yaml>toml-json-plist>ini
```
