##### 6、python执行不生成__pyche__
```shell
解决：
    执行文件头加上：
    import sys
    sys.dont_write_bytecode = True  # 不生成pyc文件
或：
python3 -B test.py         # 执行程序时加 -B参数

或在环境变量中设置如下操作：
export PYTHONDONTWRITEBYTECODE=1
```

